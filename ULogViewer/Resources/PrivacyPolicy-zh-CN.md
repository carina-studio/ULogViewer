# ULogViewer 隐私政策
 ---
+ 版本：1.4
+ 更新时间：2026/4/11

这是 ULogViewer 的隐私政策，您应该要在使用 ULogViewer 之前详细阅读本政策。 隐私政策可能会在未来有所更新，您可以在 ULogViewer 网站中查看。 当您开始使用 ULogViewer 表示您同意本隐私政策。

## 适用范围
ULogViewer 为基于开放源代码项目之软件，以下所指 ULogViewer **仅包括** 与下列页面所提供之可执行文件或压缩包内容完全相同之版本：
+ [ULogViewer 网站](https://carinastudio.azurewebsites.net/ULogViewer/)
+ [GitHub 上之 ULogViewer 项目页面及各版本发布页面](https://github.com/carina-studio/ULogViewer)

本隐私政策适用于您使用 ULogViewer 2026.0 及下一份隐私政策所指定之版本之间 (但不包括) 的所有版本。

## 个人信息的收集
我们 **不会** 收集下列的个人信息：
+ 姓名。
+ 电话。
+ 地理位置信息。

我们 **会** 收集下列可能与您个人信息有关联之数据：
+ [**IP 地址**](https://zh.wikipedia.org/wiki/IP%E5%9C%B0%E5%9D%80) (当您启用 ULogViewer 专业版后)。
+ **设备名称** (当您启用 ULogViewer 专业版后)。
+ [**MAC 地址**](https://zh.wikipedia.org/wiki/MAC%E5%9C%B0%E5%9D%80) (当您启用 ULogViewer 专业版后)。
+ **电子邮箱地址** (当您启用 ULogViewer 专业版后)。
+ 您在 ULogViewer 中打开或保存的**文件名称及路径**。
+ 您在 ULogViewer 为了日志读取而设定的 **IP 地址**。
+ 您在 ULogViewer 为了日志读取而设定的**网址 (URI)**。
+ 您在 ULogViewer 为了日志读取前进行认证而设定的**用户名及密码**。
+ 在启动 **调试模式** 时，您在 ULogViewer 主窗口中所按下的**键盘按键**。
+ 在启动 **调试模式** 时，无法符合当前日志类型所定义的任何格式之 **原始日志行 (Raw Log Line)**。
+ 应用程序错误的详细信息。
+ 您在 ULogViewer 中所输入的 **日志文字筛选词语**。
+ ULogViewer 确认是否有应用程序更新时，您的 **IP 地址**、**操作系统**及 **CPU 信息**可能会被分享至 [GitHub](https://github.com)。关于 GitHub 如何处理这些数据，请参阅 [GitHub 隐私声明](https://docs.github.com/zh/site-policy/privacy-policies/github-general-privacy-statement)。
+ 使用 ULogViewer 时的**使用数据**，包括匿名的工作会话标识符、功能使用事件（例如重新读取日志、应用日志分析、更改图表类型、日志类型操作）以及配置指标（例如日志文件个数、日志行格式数量、日志分析规则数量）。

## 个人信息的存储
+ **IP 地址** 将以 **明文** 形式存储于部署在 [Azure](https://azure.microsoft.com/zh-cn/) 之 Carina Studio 网站日志中。
+ **设备名称** 与 **MAC 地址** 将以 **密文** 形式存储于部署在 [Azure](https://azure.microsoft.com/zh-cn/) 之产品数据库中。
+ **电子邮箱地址** 将以 **密文** 形式存储于您电脑中的 ULogViewer 目录内，也将以 **明文** 形式存储于部署在 [Azure](https://azure.microsoft.com/zh-cn/) 之 Carina Studio 网站日志及产品数据库中。
+ **文件名称**、**IP 地址**及**网址 (URI)** 可能会被存储至位于您电脑中 ULogViewer 目录内的日志文件。
+ **文件名称**、**IP 地址**及**网址 (URI)** 将以 **明文** 形式存储至位于您电脑中 ULogViewer 目录内的 \*.json 文件。
+ 日志类型所需之**用户名及密码**将以 **密文** 形式存储至位于您电脑中 ULogViewer 目录内的 \*.json 文件。
+ 应用程序错误的详细信息将会被存储至位于您电脑中 ULogViewer 目录内的日志文件。
+ **日志文字筛选词语**将存储在您电脑中 ULogViewer 目录内的数据库。
+ **使用数据**将存储至 [Azure](https://azure.microsoft.com/zh-cn/)。

## 个人信息的使用
+ **IP 地址** 使用于 Carina Studio 网站之调试用途。
+ 编码之 **设备名称** 与 **MAC 地址** 用于辨识属于您所购买之 ULogViewer 专业版中已启用之设备。
+ **电子邮箱地址** 用于追踪购买与付款流程，以及辨识您已购买之 ULogViewer 专业版。
+ **日志文字筛选词语**用于在筛选日志时提供自动完成建议。
+ **使用数据**用于了解 ULogViewer 功能的使用情况，以改进应用程序。
+ 其他信息将用于调试用途 (例如分析应用程序错误)。在提供这些信息之前，您可以查看它们甚至拒绝提供这些信息。

除了政府要求之外，我们 **不会** 将您的个人信息提供给 Carina Studio 以外的人、团体及公司。
